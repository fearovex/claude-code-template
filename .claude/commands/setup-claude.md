# Setup Claude Code for Project

Analyze this project and generate a complete Claude Code configuration.

## Tasks

1. **Analyze the project**:
   - Detect tech stack (languages, frameworks, tools)
   - Identify folder structure
   - Find commands in package.json, Makefile, etc.
   - Detect architectural patterns

2. **Generate CLAUDE.md**:
   - Create CLAUDE.md file at root with:
     - Project description
     - Tech stack
     - Main commands (dev, build, test, lint)
     - Documented project structure
     - Detected code conventions
     - Important rules

3. **Suggest skills**:
   - Based on stack, suggest useful skills to create
   - Provide base structure for each suggested skill

4. **Configure basic hooks**:
   - Suggest useful hooks for the project
   - Create settings.json with recommended configuration

## Output

Show:
- Generated CLAUDE.md
- List of recommended skills with description
- Suggested hooks configuration
- Next steps to complete setup
