# Create New Skill

Create a new Claude Code skill for: $ARGUMENTS

## Process

1. **Analyze the skill topic**:
   - Understand the skill's domain
   - Identify specific knowledge needed
   - Determine relevant triggers and keywords

2. **Research best practices**:
   - Recommended patterns for the topic
   - Anti-patterns to avoid
   - Good vs bad code examples

3. **Create skill structure**:
   ```
   .claude/skills/[skill-name]/
   └── SKILL.md
   ```

4. **Generate content**:
   - Trigger-rich description
   - Specific and useful information (avoid generic content)
   - Code examples with good and bad patterns
   - Checklists or quick guides

## Skill Requirements

- Less than 500 lines
- Relevant semantic triggers
- Practical examples with code
- Project-specific information when applicable

## Output

- Create the SKILL.md file in the correct location
- Show summary of created skill
- Suggest how to test it
